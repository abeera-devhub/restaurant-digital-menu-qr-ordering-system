# Restaurant Digital Menu & QR Ordering System

A full-stack restaurant management system that allows customers to scan QR codes, browse the digital menu, place orders, track order status, and request bills digitally.

## Features

- QR Table Access
- Digital Menu
- Category Filtering
- Food Search
- Food Details View
- Cart Management
- Order Placement
- Order Tracking
- Bill Request System
- Staff Dashboard
- Admin Dashboard
- Menu Management

## Tech Stack

### Frontend
- HTML5
- CSS3
- JavaScript

### Backend
- Node.js
- Express.js

### Database
- MongoDB
